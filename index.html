About Us

We're an innovative team from Kvantorium (Dimitrovgrad) creating open technologies for developers, engineers, and makers. Our mission is to make powerful tools accessible, universal, and ready to use "out of the box."

Tech shouldn't be an elite club — it's a tool for everyone.

Our Projects

Patch

A programmable hardware automation controller with PatchWorks software. Transform complex workflows into single button presses.

  • Customizable buttons and encoders
  • Execute any Python script on press
  • Open UART-based communication protocol

PiCore Toolkit

Universal libraries (Python/C++), drivers, and API for hardware control, sensors, and network management.

Utitool

CLI utility with a unique plugin system for system operations. Install community plugins and extend functionality easily.

Nelin

Next-generation machine learning library for creating both rapid prototypes and large commercial projects.
